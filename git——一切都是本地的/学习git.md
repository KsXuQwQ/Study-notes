## 开始使用git

首先，git的优势是版本管理：记录每一次的代码变更，可以随时回退到历史版本
分支机制：允许创建独立分支进行功能开发与Bug修复
团队协作：多人可并行开发，通过合并（Merge）或变基（Rebase）集成代码
分布式架构：每个开发者都拥有完整的项目历史，支持离线工作。
