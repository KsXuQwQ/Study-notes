
## 前端环境搭建

在资料中有nginx，双击启动即可将前端界面启动，需要注意iginx启动式的存放路径需要为英文。
启动后访问 localhost:80即可访问到前端.
## 后端环境搭建

后端参用maven搭建
整体结构

| sky-take-out | maven父工程，统一管理依赖版本         |
| ------------ | ------------------------- |
| sky-common   | 子模块，存放公共类，例如：工具类，常量类，异常类等 |
| sky-pojo     | 子模块，存放实体类、VO、DTO等         |
| sky-server   |                           |
|              |                           |
